
# Loustics

Application android pour les écoliers.

Propose, après inscription et connextion, des exercices de mathématiques et de culture générale.

## Fonctionnalités développées

L'application propose de s'inscrire ou se connceter.

L'inscription demande le nom et le prénom de l'utilisateur qu'il enregistrera à l'aide de la classe Utilisateur dans la base Sugar ORM.

La connexion se fait à l'aide d'une liste (ListView) qui affiche tous les comptes créés. L'utilisateur peut choisir son compte parmi les noms existants.

Ensuite, l'utilisateur est invité à lancer un test de mathématiques ou de culture générale.

Les tests en eux-mêmes ne sont pas implémentés.
